title: 数组常用方法
---

数组方法分类：
1. **修改原数组**：`push`、`pop`、`shift`、`unshift`、`splice`、`sort`、`reverse`
2. **不修改原数组**：`concat`、`slice`、`join`、`map`、`filter`、`reduce`、`find`、`some`、`every`
3. **遍历方法**：`forEach`、`map`、`filter`、`reduce`、`find`、`some`、`every`
4. **ES6+ 方法**：`findIndex`、`includes`、`flat`、`flatMap`、`from`

`reduce` 是最强大的数组方法，可以实现 `map`、`filter`、`forEach` 等功能。
