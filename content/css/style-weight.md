question: CSS 优先级算法如何计算？
---

CSS 优先级算法以通过比较选择器的权重来决定哪个样式最终生效，同样权重下以就近原则为准。

通过以下四个等级进行叠加计算，等级高者优先（例如 0,1,0,0 总是大于 0,0,10,0）：

- 行内样式（Inline Style）：style="..."，权重为 1,0,0,0
- ID 选择器：如 #header，权重为 0,1,0,0
- 类、伪类、属性选择器：如 .class、:hover、[type="text"]，权重为 0,0,1,0
- 元素和伪元素选择器：如 div、p、::before，权重为 0,0,0,1

补充规则：

- !important：属性后加 !important 会无视规则优先级，拥有最高权重。
- 通配符与继承：* 通配符、+、> 符号不增加权重。继承的样式优先级最低，甚至低于通配符选择器。
- 层叠顺序：若优先级完全相同，则遵循“后加载覆盖先加载”的原则，即后定义的样式生效。 

**同权重: 内联样式表> 嵌入样式表> 外部样式表**
